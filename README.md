## SOBRE

Projeto Landing-page de estudo desenvolvido em Angular 18 com SSR.

## PRÉ-REQUISITOS

- NodeJS
- Git

## CLONANDO PROJETO

```
git clone https://github.com/andersonjuniorz/Sabor-E-Arte.git
cd Sabor-E-Arte
```

Instalar e rodar as dependências:
```
npm install
npm start
```
